## Installation

1. From the project root directory:
```sh
cd rotavo/
```
2. Install the dependencies
```sh
npm install
```
3. Build the bundle
```sh
npm run build
```
4. Serve the `public` directory using the web server of your choice. The project is configured for Firebase hosting
```sh
firebase serve
```
